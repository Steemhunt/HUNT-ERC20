# HUNT Token
Temporary HUNT token contract using ERC20 standard

## Dev Note
**You need to run ganache-cli locally before testing**

```bash
cd hunt
npm install && npm run test
npm install -g truffle # 5.0.2
```
> Deploy
```bash
truffle migrate --network ropsten
```
